# Examples

## Purpose

The `examples` directory contains reference implementations, sample configurations, API examples, prompt examples, and end-to-end workflows that demonstrate how to use XeniosAI.

Examples are educational artifacts and should represent best practices without replacing production documentation.

## Planned Contents

- API Request & Response Examples
- AI Tool Usage Examples
- Sample Property Configurations
- Deployment Examples
- Prompt Examples
- Integration Examples
- End-to-End Guest Journey Examples
- Booking Workflow Examples

## Design Principles

- Examples should be complete and runnable where practical.
- Keep examples synchronized with the latest APIs and schemas.
- Prefer realistic hospitality scenarios.
- Avoid embedding secrets or production credentials.
- Examples should be easy for both humans and AI agents to understand.
